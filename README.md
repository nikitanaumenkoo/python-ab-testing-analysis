# A/B Testing Analysis  
**Python · Statistics · Google Colab · Data Visualization**

## Опис проєкту
Проєкт присвячений аналізу результатів A/B-тестування з використанням статистичних методів у Python.  
Мета — оцінити статистичну значущість ключових конверсійних метрик та підготувати дані для подальшої візуалізації.

## Інструменти
- Python (Google Colab)
- Pandas, NumPy, SciPy
- Статистичні тести для A/B-аналізу

## Основні кроки
- формування датасету з кількох таблиць у BigQuery
- розрахунок конверсійних метрик у тестових і контрольних групах
- автоматизований розрахунок статистичної значущості через цикли та масиви
- формування фінального датасету для візуалізації результатів

## Проаналізовані метрики
- add_payment_info / session
- add_shipping_info / session
- begin_checkout / session
- new_accounts / session

## Особливості реалізації
- розрахунок статистичної значущості без жорсткого закріплення кількості метрик
- можливість аналізу в різних розрізах (тест, країна, пристрій тощо)
- фінальний результат підготовлений для використання у дашбордах

## Результат
Отримано універсальний Python-скрипт для A/B-аналізу, який дозволяє масштабувати кількість метрик та застосовувати статистичні тести до різних сегментів даних.
